The basics

html + css + js

  • html + css + js are the three technologies that make up the "front-end"
  • these technologies advance over time but haven't been replaced since their introduction
  • packages like Bootstrap or jQuery are one or more of these technologes combined

html

  • the structure to map content to
  • html is like a tree with parent/child relationships


css

  • applies style to your html

js

  • gives us page functionality

Media queries

  • defines multiple styles based on device specifics like orientation, display dimensions & resolution

Project Planning: setting responsive breakpoints

  • display sizes are NOT standardized
  • display sizes are a spectrum and breakpoints are points along that spectrum where you want to redefine your UI and layout
  • typically you set 3 breakpoints (mobile, tablet, desktop) and let the devices sort themselves out based on their dimensions

Resolution and device orientation

  • screen resolution and device orientation add complexity to our nice breakpoint spectrum
